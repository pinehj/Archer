1. 작업하신 브랜치가 최신화되었나요? (behind 커밋이 존재하지 않나요?)
2. 추가하신 기능 및 전체 로직이 정상적으로 동작하는 것을 확인하셨나요? (기능 테스트를 완료하셨나요?)
3. Assignee에 본인 계정을 추가하고, labels에 본인 이름 label과 PR의 작업내용을 잘 나타낼 수 있는 라벨을 추가하셨나요?
